## Set Up ##

1. Install Node.js - https://nodejs.org/en/
2. Run npm install

## Current Commands ## 
/updatePrices - Updates prices in mongoDB from https://metals-api.com (can only be run once per day)
/prices - Displays latest prices held in mongoDB
